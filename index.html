🛡 Beheer accounts

Wijs rollen toe aan accounts. Alleen jij (super-user) ziet dit scherm.
Rollen uitgelegd:
  • User — kan alleen taken afvinken
  • Admin — kan ook taken bewerken/toevoegen/verwijderen, plannen importeren en wijzigingen doorvoeren in Versiebeheer
